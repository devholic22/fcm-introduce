# fcm-introduce-spring
본 리포지터리는 블로그 내용을 쉽게 실습하실 수 있도록 스프링 환경에서의 FCM 알림 기능 구현 코드를 담은 리포지터리입니다.

## 적용한 기술 환경
* Spring Boot 3.3.2
* Java 17
* Spring Data JPA 3.3.2
* Hibernate 6.5.2
* Spring Data Redis 3.3.2
* Spring Web 3.3.2
* Lombok 1.18.34
* MySQL 8.3.0
* FCM (firebase-admin) 9.3.0
